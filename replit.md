# Export & Import Co.

A React (Create React App) marketing website for an agricultural export company based in Bengaluru, Karnataka. It showcases products (turmeric fingers, turmeric powder, premium rice), certifications, and a contact inquiry form.

## Stack

- **React 19** (Create React App)
- No backend — fully static single-page app
- All product images are bundled locally in `src/`

## How to run

The app runs on port 5000 via the "Start application" workflow:

```
PORT=5000 BROWSER=none npm start
```

No environment variables or external services required.

## Project structure

- `src/App.js` — entire app (all pages: Home, Products, Certifications, Contact)
- `src/App.css` — all styles
- `src/*.jpeg / *.jpg` — product images (turmeric fingers, turmeric powder, rice)
- `public/` — static assets (favicon, manifest)
