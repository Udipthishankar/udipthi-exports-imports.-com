# Export & Import Co. — Premium Agro Exports

A responsive React website for an agricultural export company based in Bengaluru, Karnataka, India. The site showcases the company's products, certifications, and contact details for international buyers.

## Pages

- **Home** — Hero banner, company mission, feature highlights (smart sourcing, hygienic processing, global supply chain), and key stats
- **Products** — Image sliders for Turmeric Fingers, Turmeric Powder, and Premium Rice with inquiry CTA
- **Certifications** — APEDA Registration, ISO 22000:2018, and FSSAI Quality Seal badges
- **Contact** — Split-layout contact panel with inquiry form

## Tech Stack

- **React 19** (Create React App)
- Pure CSS — no UI library
- Fully static SPA — no backend required

## Getting Started

```bash
npm install
npm start        # dev server on http://localhost:3000
npm run build    # production build → /build
```

## Deployment

Configured for **static deployment** on Replit. The build command is `npm run build` and the output directory is `build/`.
